## Variables and Constants

### let and const

```javascript
const message = "Hello World!";
console.log(message);

const hobbies = ["Sports", "Cooking", "Reading"];
hobbies.push("Working"); // This works - you can mutate the array
// hobbies = []; // This would fail - you can't reassign const

console.log(hobbies);
```

<Warning>
Use `const` by default. Only use `let` when you know the variable will be reassigned.
</Warning>

## Functions

### Function Declarations and Expressions

<CodeGroup>

```javascript Regular Function
function createGreeting(userName, message = "Hello!") {
  return "Hi, I am " + userName + ". " + message;
}

const greeting1 = createGreeting("Max");
console.log(greeting1); // "Hi, I am Max. Hello!"

const greeting2 = createGreeting("Manuel", "Hello, what's up?");
console.log(greeting2); // "Hi, I am Manuel. Hello, what's up?"
```

```javascript Arrow Function
const createGreeting = (userName, message = "Hello!") => {
  return "Hi, I am " + userName + ". " + message;
};

// Shorthand for single expression
const greet = (name) => "Hello, " + name;
```

</CodeGroup>

<Tip>
Arrow functions are commonly used in React for event handlers and callback functions.
</Tip>

## Objects and Classes

### Object Literals

```javascript
const user = {
  name: "Max",
  age: 34,
  greet() {
    console.log("Hello!");
    console.log(this.age);
  }
};

console.log(user.name); // "Max"
user.greet();
```

### Classes

```javascript
class User {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }

  greet() {
    console.log("Hi!");
  }
}

const user1 = new User("Manuel", 35);
console.log(user1);
user1.greet();
```

## Arrays and Array Methods

### Common Array Operations

<Tabs>
  <Tab title="map()">
    Transform each element in an array:
    
    ```javascript
    const hobbies = ["Sports", "Cooking", "Reading"];
    const editedHobbies = hobbies.map((item) => ({ text: item }));
    console.log(editedHobbies);
    // [{ text: "Sports" }, { text: "Cooking" }, { text: "Reading" }]
    ```
  </Tab>
  
  <Tab title="findIndex()">
    Find the index of an element:
    
    ```javascript
    const hobbies = ["Sports", "Cooking", "Reading"];
    const index = hobbies.findIndex((item) => item === "Sports");
    console.log(index); // 0
    ```
  </Tab>
  
  <Tab title="push()">
    Add elements to an array:
    
    ```javascript
    const hobbies = ["Sports", "Cooking"];
    hobbies.push("Working");
    console.log(hobbies); // ["Sports", "Cooking", "Working"]
    ```
  </Tab>
</Tabs>

<Note>
In React, you'll use `map()` frequently to render lists of components.
</Note>

## Destructuring

### Array Destructuring

```javascript
const [firstName, lastName] = ["Max", "Schwarzmüller"];

console.log(firstName); // "Max"
console.log(lastName); // "Schwarzmüller"

// Instead of:
// const firstName = userNameData[0];
// const lastName = userNameData[1];
```

### Object Destructuring

```javascript
const { name: userName, age } = {
  name: "Max",
  age: 34
};

console.log(userName); // "Max"
console.log(age); // 34

// Instead of:
// const userName = user.name;
// const age = user.age;
```

<Tip>
Object destructuring is essential for working with props in React components.
</Tip>

## Spread Operator

### Array Spreading

```javascript
const hobbies = ["Sports", "Cooking"];
const newHobbies = ["Reading"];

const mergedHobbies = [...hobbies, ...newHobbies];
console.log(mergedHobbies); // ["Sports", "Cooking", "Reading"]
```

### Object Spreading

```javascript
const user = {
  name: "Max",
  age: 34
};

const extendedUser = {
  isAdmin: true,
  ...user
};
console.log(extendedUser);
// { isAdmin: true, name: "Max", age: 34 }
```

<Warning>
The spread operator creates a shallow copy. Nested objects are not cloned deeply.
</Warning>

## Control Structures

### Conditional Statements

```javascript
const password = prompt("Your password");

if (password === "Hello") {
  console.log("Hello works");
} else if (password === "hello") {
  console.log("hello works");
} else {
  console.log("Access not granted.");
}
```

### Loops

```javascript
const hobbies = ["Sports", "Cooking"];

for (const hobby of hobbies) {
  console.log(hobby);
}
```

## Modules (Import/Export)

<CodeGroup>

```javascript util.js (Named Exports)
export const apiKey = "abc123";
export const abc = "content";
```

```javascript app.js (Named Imports)
import { apiKey, abc as content } from "./util.js";

console.log(apiKey);
console.log(content);

// Or import everything:
import * as util from "./util.js";
console.log(util.apiKey);
```

```javascript util.js (Default Export)
export default function greet(userName) {
  return "Hello " + userName;
}
```

```javascript app.js (Default Import)
import greet from "./util.js";

console.log(greet("Max"));
```

</CodeGroup>

## Asynchronous Code

### setTimeout and Callbacks

```javascript
function handleTimeout() {
  console.log("Timed out!");
}

const handleTimeout2 = () => {
  console.log("Timed out ... again!");
};

setTimeout(handleTimeout, 2000);
setTimeout(handleTimeout2, 3000);
setTimeout(() => {
  console.log("More timing out...");
}, 4000);
```

### Function Scope and Closures

```javascript
function init() {
  const message = "Hello";
  
  function greet() {
    console.log(message); // Can access outer scope
  }

  greet();
}

init();
```
